# Registeration
인프런 - 안드로이드 스튜디오 / 수강신청 도우미 

## 개발환경 및 사용언어
> Android-studio(Java)    
> Cafe24

## 구현내용
- 로그인/회원가입(아이디중복체크)/로그아웃    
-  공지사항, 강의목록, 강의분석, 강의삭제    
-  시간표, 학점제한    
-  강의 통계 확인(인기순위, 성별분포, 학과별 분포등)    
-  어플리케이션 정보(팝업창)    
-  비밀번호 암호화
